🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Backend won't start?
   → Make sure venv is activated
   → Check Python version (3.11+ required)

❌ Frontend shows connection error?
   → Ensure backend is running on port 5000
   → Check backend/.env has correct settings

❌ Database error?
   → Delete inventory.db file
   → Run: python seed.py again

❌ AI chat not working?
   → AI chat requires ANTHROPIC_API_KEY
   → All other features work without it
